# Garabata font

Place your Garabata font files here so the site can load them:

- **Garabata.woff2** (required for modern browsers)
- **Garabata.woff** (optional fallback for older browsers)

The project is already set up to use Garabata as the main font in `src/index.css`. If the files are missing, the site will fall back to the system font.
